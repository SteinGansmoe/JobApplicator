JobApplication Tracker Tool. 

### 🧩 1. Project overview 

JobTracker is a web application that allows users to track job applications through different stages such as saved, applied, interview, and offer.

### 🧩 2. Features

- Add and manage job applications
- Update application status
- View all applications in a structured list
- Dashboard with real-time application statistics
- Responsive layout with dark mode support

### 🧩 3. Tech stack

- Next.js
- React
- TypeScript
- Tailwind CSS

### 🧩 4. Development approach

- Built the initial MVP manually to understand core functionality
- Used Codex for small, controlled refactors
- Broke down work into GitHub issues for better task management

### 🧩 5. AI usage

AI tools like Codex were used as a development assistant, not a replacement.
All changes were carefully reviewed and understood before being applied.


### 🧩 6. Learning focus

This project is built with a strong focus on learning and understanding.
Even when using Codex for certain tasks, I ensure that I fully understand all changes before applying them.

If I cannot explain a piece of code, it will not be included in the project.

### 🧩 7. Future improvements

- Improve overall UI/UX (current version is MVP-focused)
- Add filtering by application status
- Improve UI/UX for application cards
- Integrate backend (e.g. Supabase) for persistent data
- Add search functionality
